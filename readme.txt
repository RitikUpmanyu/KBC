Description of how to run the code and observe the output:
1. The exe is called 2019B3PS517_P3.exe. It is a console Application.
	 PLEASE RUN THE CONSOLE IN FULL SCREEN or enough width otherwise things might not display properly.
2. The input files are -> 1. questions.txt used for reading questions and options
		    2. features.c used for displaying questions, options, money and implementing lifelines.
		    3. formatting.c used for displaying boxes around questions, options and money. 
3. count.txt will automatically be generated when you compile the code for the first time, 
	it will keep track of how many times the game has been played. So that, we can display alternate questions based on count. 
4. When you run the exe, it will prompt display the first question on the screen. along with your lifelines and a money board.
5. For the first 4 questions, You will only have 30 seconds to answer.
6. You can enter a/A/b/B/c/C/d/D to choose a option or you can take lifeline if they are still remaining-
	1. Enter l/L for 50-50.
	2. Enter f/F for Flip-the-question.
7. You can even quit at any point by entering q/Q. After you loose the game you can play again by entering p/P.
